Configuración y publicación

Los QR solo funcionan si apuntan a una URL pública accesible.

Esta versión añade un campo de URL pública base para que el QR abra una página real desde cualquier móvil. Puedes pegar ahí la URL donde alojes el HTML, por ejemplo en Netlify, GitHub Pages o tu propio dominio.

Si el archivo se abre solo como archivo local o desde un enlace no público, el QR puede generarse, pero no llevará a una página accesible desde otro dispositivo.

Debe ser la dirección pública exacta donde está publicado este archivo HTML.

Se generará un QR por cada acceso y uno adicional para ver el total.
