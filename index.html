Configuración guiada

La versión más simple con Supabase y polling.

Solo necesitas crear una tabla en Supabase, pegar la URL del proyecto y la publishable key, y luego la app crea el evento, genera los QR y actualiza el panel cada 1,5 segundos.

Paso 1. En Supabase crea un proyecto y abre el SQL Editor.
Paso 2. Pega el SQL de abajo y ejecútalo una sola vez.
Paso 3. Copia la Project URL y la Publishable/anon key, pégalas abajo y genera el evento.

No uses la service role key en esta app.
